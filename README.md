## Описание проекта

Чат для общения в реальном времени.

## Установка

- `npm install` - установка зависимостей.
- `npm run dev` - запуск проекта в режиме разработки.
- `npm run build` - запуск сборки проекта.
- `npm run start` - запуск проекта на сервере.
